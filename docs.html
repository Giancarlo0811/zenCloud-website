Instalar CLI

Requerimientos

  • Windows 10, Mac OSX, Linux
  • Node.js v12 o en adelante

Instalar

Mac (Homebrew)

$ brew install zenCloud-cli

NPM

$ npm install zenCloud-cli

Yarn

$ yarn install zenCloud-cli
